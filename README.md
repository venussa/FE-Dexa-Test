## REQUIREMENT
1. Node JS
2. Firebase

## APPLICATION SETUP
1. git clone https://github.com/venussa/FE-Dexa-Test.git
2. copy `env.example` to `.env`
3. set API base url and firebase config in `.env` (you can get it from firebase console)
4. npm run `npm install`
5. npm run predev
6. npm run dev
